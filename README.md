Project for OOP 24/25 course.
A 2D-HorizontalScrolling Videogame inspired by GeometryDash and bash terminals aestethic.
Developed in java

## How to run
`java -jar Geometry-Bash-all.jar`

Digit `help` or `commands` in the main menu to show the list of available commands to use the application.

Digit `start` to start the game.

Press `esc` on the keyboard to access pause menu.
